Final Project

I’ve always loved the outdoors and it’s been an escape for me. Unfortunately our ecosystem is dying and there are many foundations doing their best to try and save it. For my project I would like to demonstrate the importance of the outdoors and the foundations that support them.
My plan for this project is to create an interactive journey into a forest. Different aspects of the forest will be able to be clicked and be educational. Some of these clickables will be links to various youtube videos from these foundations to further infrom the user.
Another idea I had was to use Google Earth for users to explore the worlds forests.
I wanted to create the atmosphere using a illustrative style similar to these.
I hope to use a hover effect that generates noises over different divs in the website. The noises would be various animals and eco systems within the forest.
